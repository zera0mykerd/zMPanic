# Privacy Policy for zM SOS GUARD

**Last updated: January 2026**

zM SOS GUARD is an Open Source personal security application. This privacy policy explains how we handle your data.

### 1. Data Collection
zM SOS GUARD **does not collect, store, or share** any personal data with the developer or any third-party company. All data remains under the user's total control.

### 2. Permissions & Usage
The app requires the following permissions to function:
* **Camera:** To record video evidence during an SOS event.
* **Microphone:** To record audio during an SOS event.
* **Location:** To attach GPS coordinates to the recorded files.
* **Storage/Media:** To temporarily store video segments before uploading.

### 3. Data Transmission
All video and location data are transmitted directly to a **user-defined server IP address** via HTTP/HTTPS. The developer of zM SOS GUARD has no access to this data, as it is sent directly to your own infrastructure.

### 4. Third-Party Services
This app does not use any third-party analytics, tracking, or advertising SDKs.

### 5. Open Source
The source code is public and transparent. You can audit the code at: https://github.com/zera0mykerd/zMPanic

### 6. Contact
If you have any questions, you can open an issue on the official GitHub repository.
